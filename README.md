# Shop - E-Commerce Application

![Shop Icon](public/shop_icon.png)

A modern, responsive e-commerce web application built with Angular, featuring product browsing, cart management, wishlist functionality, and a seamless checkout process.

## Features

- **Product Catalog**: Browse and search through a wide range of products with detailed views.
- **Shopping Cart**: Add, remove, and manage items in your cart with quantity controls.
- **Wishlist**: Save favorite products for later.
- **User Authentication**: Sign-in and sign-up dialogs for user management.
- **Checkout Process**: Multi-step checkout with shipping and payment forms.
- **Order Summary**: Review and confirm orders before completion.
- **Responsive Design**: Optimized for desktop and mobile devices using Tailwind CSS.
- **State Management**: Efficient state handling with NgRx Signals.

## Technology Stack

- **Framework**: Angular
- **UI Library**: Angular Material 
- **Styling**: Tailwind CSS
- **State Management**: NgRx Signals 
- **Language**: TypeScript
- **Build Tool**: Angular CLI 
- **Other**: RxJS, Immer for immutable updates

## Prerequisites

Before running this project, ensure you have the following installed:

- Node.js (version 18 or higher)
- npm (comes with Node.js)
- Angular CLI (install globally with `npm install -g @angular/cli`)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DeveloperRucel07/shop.git
   cd shop
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the development server:
   ```bash
   npm start
   ```
   The application will be available at `http://localhost:4200`.

2. Build for production:
   ```bash
   npm run build
   ```
   The build artifacts will be stored in the `dist/` directory.

3. Run tests:
   ```bash
   npm test
   ```
